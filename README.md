## awesome config

### shortcuts

all shortcuts for awesome wm can be found in [~/.config/awesome/shortcuts.lua](.config/awesome/shortcuts.lua)

#### mod keys

- **mod4** = super key / window key
- **mod1** = alt key

#### hotkeys

- browser ( **brave-dev** ) = mod4 + control + b
- terminal ( **alacritty** ) = mod4 + t
- **close apps** = mod4 + w
- **show help** = mod4 + s
- **dmenu** = mod4 + d
- **mount usb** = mod1 + m
